## Antigravity Build Instructions

You are building a v1 internal CRM UI using Next.js App Router, TypeScript, Tailwind, and shadcn/ui.

Rules
- Follow the existing folder structure exactly
- Do not create new folders unless explicitly asked
- Do not build integrations or backend logic yet
- Use mock data where needed
- Focus on UI and layout first
- Keep components small and reusable
- Prefer composition over complexity

UI Guidelines
- Clean, minimal, premium
- Neutral colors (white, gray, black, beige)
- Inspired by Linear, Notion, Stripe dashboards
- Desktop-first, responsive second

Scope for v1
- Dashboard layout (sidebar + topbar)
- Properties list page
- Property detail page
- Leads inbox page
- Deals pipeline view

Output Rules
- One feature at a time
- One screen per response
- No placeholder text like “lorem ipsum”
- No overengineering
